Introduction¶

Coronary artery disease (CAD) is a disease of the heart in which patients are at high risk for a heart attack. CAD is caused by buildup in the walls of the arteries such as cholesterol, over time this can lead to a blockage partially blocking the blood flow (Centers for Disease Control and Prevention, 2021; Cleveland Clinic, 2022). In this study, we will be using a mixture of census data and medical data to predict the diagnosis of CAD.

The question that we are answering with this dataset is; Is it possible to accurately predict CAD from only census data, or only medical data? Is it more accurate to combine both data types in a single model?

To perform this analysis, we will be using the Cleveland Clinic Data of the Heart Disease Data. This data set looks at 303 patients diagnosed with CAD via an invasive angiogram and the results of noninvasive clinical tests which could indicate CAD (Detrano et al., 1989).

Methods¶

The columns to be used in our analysis are as follows: for census data, sex and age; for medical data, resting blood pressure (trestbps), cholesterol level (chol), fasting blood sugar (fbs) and ST depression (oldpeak). The state of diagnosis (num) will be used.

We will build three classification models: the first using census data; the second, medical data; and the third, both. We will then evaluate the accuracy of all three models and compare metrics, determining which set of variables results in the best prediction accuracy.

We will be using the K-nearest neighbors classification algorithm to predict observations from our training set. We will complete each of the 3 models by following these steps.

  1. Randomly divide the data into training and test data sets to find the best K value, utilizing the training set to create the classifier and the test set to evaluate its accuracy. We will tune the classifier by optimizing estimated accuracy using cross-validation.
  2. Create a model specification for K-nearest neighbors, specifying the number of neighbors.
  3. Create a centering/scaling recipe using the recipe function and specify target and predictor variables.
  4. Pass the model specification and recipe under a workflow, specifying the data to run on.
  5. Make a prediction for the new observation using predict function by passing the fit object and the new observation into it.
  6. Collect the metrics of our predictions, specifying the accuracy.

Data Visualization: We intend to plot a bar graph with accuracy data for each model on the y-axis to visualize whether it is more accurate to look at census data, medical data, or both

Expected Outcomes and Significance¶

Existing literature shows a positive relationship between CAD and age (Brown et al., 2022), resting blood pressure (Cleveland Clinic, 2021), cholesterol level (Centers for Disease Control and Prevention, 2021), fasting blood sugar (American Heart Association Inc., 2021), ST depression (Klabunde, 2023); and the male sex (Gheisari, 2020). The expected outcome of this classification analysis is that census data will not be sufficient to predict CAD, whereas medical data will be. The combined model will be most accurate in CAD prediction.

The findings from this analysis could be insightful into how medical data and census data could be used to provide accurate prediction models for CAD risk across a general population. If the demographic predictors are sufficient to predict CAD, diagnosis could be more accessible.

This classification analysis could lead to further questioning on how CAD diagnosis could require less medically-invasive tests by using only demographic and data of personal history.
